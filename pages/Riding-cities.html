Vue d'ensemble du projet

Sur base de la maquette représentant la nouvelle page de l’association, je devais mettre à jour la page Internet de Riding Cities, une association sportive qui promeut le skate en région :

Ma mission pour ce projet était de :
● Créer une nouvelle section, “Les membres fondateurs”, contenant du texte et des images en colonnes avec les noms et photos des créateurs de l’association.
● Modifier la section comprenant le bouton “Télécharger le planning de cours” pour ajouter un titre et deux boutons permettant de télécharger deux plannings de cours différents (un pour les enfants et un pour les adultes).
● Rattacher les plannings de cours pour enfants et pour adultes aux boutons de téléchargement.

Les compétences évaluées étaient d'intégrer des éléments sur une page web en utilisant les fondamentaux de HTML

Pour mener à bien ce projet, j'ai suivi les cours OpenClassrooms suivants :
● Mettez en place votre environnement
● Créer votre site web avec HTML5 et CSS3

N’hésitez pas à consulter le code et le visuel du projet en visitant les liens correspondants ci-dessous.
