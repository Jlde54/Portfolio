Vue d'ensemble du projet

Ma mission ne consistait pas à développer un site mais à mener, préparer et planifier le projet de développement d’un site, ainsi qu'à mettre en place une méthodologie de travail agile et en collaboration avec notre Product Owner.

Les étapes du projet :
● Parcourir le document de spécifications fonctionnelles, les maquettes et les user stories.
● Visualiser les maquettes Figma en mode prototype afin de bien comprendre les différents enchaînements des écrans.
● Mettre en place une veille technique dont les sources doivent donc être en lien avec le projet.
● Identifier les besoins techniques.
● Choisir et configurer un outil de gestion de projet (Wakelet).
● Découper le projet en différentes tâches pouvoir coder le projet Menu Maker by Qwenta.
● Estimer les différentes tâches en fonction de la taille.
● Préparer les supports de présentation pour le Product Manager et pour le client.

Le projet se basait sur des éléments préparés au préalable :
● la maquette desktop conçue par notre UI designer, qui présente le résultat final attendu du site.
● le document de spécifications fonctionnelles qui liste les exigences qui devront être prises en compte dans le développement du site.
● le tableau des user stories sur Notion qui présente les besoins et attentes des utilisateurs du site.
● un modèle de spécifications techniques à remplir.

Les compétences évaluées étaient :
● Présenter la solution technique
● Rédiger les spécifications techniques d'un projet web à partir de besoins fonctionnels
● Découper une fonctionnalités en tâches pour préparer le développement
● Mettre en place une méthode de veille technologique
● Suivre le déroulement du projet grâce à un outil de gestion de projet

Pour mener à bien ce projet, j'ai suivi les cours OpenClassrooms suivants :
● Initiez-vous à la gestion de projet agile
● Gérez votre projet informatique facilement
● Mettre en place un système de veille informationnelle
● Devenez développeur agile
● Appliquez le principe du Domain-Driven Design à votre application
● Réalisez un cahier des charges fonctionnel
● Créez une maquette pour le développement web
● Créez une maquette web avec Figma
● Donnez une présentation en entreprise
