Vue d'ensemble du projet

J'avais en charge de dynamiser le site Internet statique de l'imprimerie Print it. Ce projet avait pour but de découvrir le langage JavaScript.

Les étapes du projet :
● Mettre à jour le code HTML afin de pouvoir avoir un carrousel dynamique.
● Ajouter des Event Listeners sur les flèches afin de pouvoir gérer le changement des images du carrousel.
● Ajouter des bullet points au slider pour savoir sur quelle slide on se trouve.
● Modifier le slide au clic sur le bouton (image + bullet point).
● Mettre en place le défilement infini sur le carrousel.

Les compétences évaluées étaient :
● Programmer en utilisant les fondamentaux de JavaScript

Pour mener à bien ce projet, j'ai suivi les cours OpenClassrooms suivants :
● Créer votre site web avec HTML5 et CSS3
● Apprendre à programmer avec JavaScrip
● Découvrir le fonctionnement des algorithmes
